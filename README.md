# zirui-wang.com

Personal academic website of Zirui Wang, Finance Ph.D. student at the
University of Texas at Austin, McCombs School of Business.

Static site (plain HTML/CSS/JS, no build step) served via GitHub Pages.

- `index.html` — the whole site
- `assets/css/site.css` — styles
- `assets/js/site.js` — interactions (galleries, comments carousel, menu)

To preview locally, run any static server from the repo root, e.g.:

```
python -m http.server 8765
```

then open <http://localhost:8765>.
